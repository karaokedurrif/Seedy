#### `/scripts/benchmark-vllm.sh`
**Función:** Benchmarking completo con 4 tests

**Tests implementados:**
1. **Simple Query** — 10 runs de "2+2=" (latencia promedio)
2. **Code Generation** — 5 runs de factorial function (latency + tokens generados)
3. **Concurrent Requests** — 10 requests paralelos (throughput)
4. **Long Context** — 8K tokens input (prefill performance)

**Métricas capturadas:**
- Latency P-avg para cada test
- Tokens generated
- Throughput (req/s)
- GPU memory delta
- Cache hit rate (from logs)
- Score vs baseline (% improvement)

**Resultados guardados en:** `~/seedy/benchmark_results/benchmark_YYYYMMDD_HHMMSS.txt`

**Uso:**
```bash
~/seedy/scripts/benchmark-vllm.sh
```

---

#### `/scripts/monitor-vllm-health.sh`
**Función:** Health check automático con alerting

**Checks implementados:**
1. Health endpoint (HTTP 200)
2. Container status (docker inspect)
3. Container uptime
4. GPU memory usage
5. Inference test (2+2 query)

**Alerting:**
- Exit code 1 si algún check falla
- Log detallado para troubleshooting
- Timestamp de cada check

**Uso:**
```bash
~/seedy/scripts/monitor-vllm-health.sh

# Automatizar con systemd:
sudo cp /tmp/vllm-healthcheck.service /etc/systemd/system/
sudo cp /tmp/vllm-healthcheck.timer /etc/systemd/system/
sudo systemctl enable --now vllm-healthcheck.timer
```

## 🚨 Known Issues & Mitigation

### Issue 1: GPU Memory N/A en nvidia-smi
**Causa:** Unified memory ARM64 (GB10) no reporta separate GPU memory  
**Impacto:** Cosmético, no afecta performance  
**Mitigation:** Usar `docker stats vllm-coder` para estimar uso real

### Issue 2: Baseline benchmark latencia alta
**Causa:** Ollama qwen2.5:72b cargado durante benchmark (contention)  
**Impacto:** Baseline no refleja best-case vLLM solo  
**Mitigation:** Ejecutar benchmark con Ollama idle (unload 72b)

### Issue 3: Concurrent throughput muy bajo (0.04 req/s)
**Causa:** Max 8 concurrent → serialización de requests  
**Impacto:** Benchmarkバottleneck, no refleja capacidad real  
**Mitigation:** Balanced config aumenta a 12 concurrent (esperado +150%)
